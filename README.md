## Описание программы

Currency Converter — это простое GUI-приложение для конвертации валют с использованием внешнего API.  
Программа позволяет быстро получать актуальные курсы валют, выполнять конвертацию и сохранять историю операций.

---

## Требования для запуска

- Python 3.10 и выше
- Tkinter
- Библиотека requests
- JSON
- Git и GitHub

---

## Как запустить

1) Установить зависимости:
pip install requests

2) Запустить программу:
python main.py

---

## Используемое API. Как получить API ключ

Программа использует сервис:
https://www.exchangerate-api.com/

Чтобы получить ключ, нужно:
1. Перейти на сайт: https://www.exchangerate-api.com/
2. Зарегистрироваться
3. Получить бесплатный API ключ
4. Вставить ключ в файл "main.py" в переменную Python:
API_KEY = "ВАШ_API_КЛЮЧ"

---

## Пример использования

1) Выбрать валюту: USD → EUR
2) Ввести сумму: 100
3) Нажать "Конвертировать"
4) Получить результат и запись в историю
